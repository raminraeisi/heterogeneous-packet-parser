Detailed Description

template<class NewPacket, cdp::globals::PacketIDType ID, class _First, class ... _Rest>
class cdp::packet::PacketTemplate< NewPacket, ID, _First, _Rest >

: A template based class which is derived from PacketBase to add the following capabilities 1- Forming the internal packet structure using std::tuple 2- Auto registration of the derived packet classes in the PacketFactory 3- Auto correcting the endianness Actually we have an internal buffer holding whole data of the packet and the tuple will be holding the typed pointers to the fileds in the buffer

Template Parameters
NewPacketrefers to the new packet class to be defined
IDdesired new packet ID
_FirstPointer type to the first field of the packet
_Rest...Pointer type to the rest fields of the packet
